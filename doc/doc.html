Asset Loading

loadSprite()loadSound()

Query

width()height()time()dt()

Scene

scene()go()reload()

Objects

sprite()rect()text()destroy()destroyAll()

Group Events

action()collide()click()
loadSprite(name, src, )

load a sprite

loadSound(name, src, )

load a sound

width()

canvas width

height()

canvas height

time()

current game time

dt()

delta time since last frame

scene(name, )

start describing a scene

go(name, )

switch to a scene

reload(name, )

reload a scene, reinitialize all states

sprite(id, conf, )

add a sprite to scene

rect(width, height, conf, )

add a rect to scene

text(str, conf, )

add a rect to scene

destroy(obj, )

remote an obj from scene

destroyAll(tag, )

destroy all objects that has the specified tag

action(tag, cb, )

run callback every frame for all object with the specified tag

collide(tag, cb, )

run callback for every collision between all object with the specified tag

click(tag, cb, )

run callback for every click on all object with the specified tag
